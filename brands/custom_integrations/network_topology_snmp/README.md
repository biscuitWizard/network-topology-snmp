# Network Topology SNMP brand assets

These assets are original, procedurally generated artwork for the Network Topology SNMP Home Assistant custom integration. They intentionally avoid copyrighted vendor logos and use a simple network-node and SNMP signal motif suitable for small icon sizes.

The PNG files were generated with a standard-library Python raster script during project setup. They may be used as source assets for a later `home-assistant/brands` submission, subject to that repository's review requirements.
